# Mirror Networking (UPM Fork)

This is an automated fork of [Mirror](https://github.com/mirrornetworking/mirror) specifically structured for the **Unity Package Manager (UPM)**.

## Installation via Git URL

1. Open the **Package Manager** in Unity (Window > Package Manager).
2. Click the **+** button and select **Add package from git URL...**
3. Paste the following URL:
   `https://github.com/Chrisdbhr/Mirror.git#upm`

## Why use this fork?
- **Lightweight:** Downloads only the essential `Assets` folder.
- **Automated Updates:** Synchronized with the official Mirror releases.
- **Native UPM Support:** Correct folder structure and metadata.
